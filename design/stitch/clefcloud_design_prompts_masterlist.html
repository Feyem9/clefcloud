ClefCloud - Design Prompts Masterlist 🎵☁️ 1. Home Page (Landing) "Generate a high-converting landing page for 'ClefCloud', a digital sheet music platform. Include a hero section with a catchy headline like 'Vos partitions, partout, tout le temps', a search bar mockup, a 'How it works' section with 3 steps (Upload, Organize, Play), and a testimonial section. Use a clean, modern aesthetic with a 'Music-first' vibe. Ensure there are clear Call-to-Action (CTA) buttons for 'Commencer gratuitement'." 2. Library (Partitions) "Design a 'My Library' page for ClefCloud. Feature a dense but organized grid of sheet music cards. Each card should show a thumbnail of the first page, the title, composer, and a tag for the category (e.g., 'Chant liturgique'). Add a robust sidebar with filters for 'Compositeur', 'Thème', and 'Favoris'. Include a 'Tri par date' dropdown and a search-as-you-type bar at the top." 3. Upload Page "Create a modern 'Upload Partition' interface. Use a large drag-and-drop zone with a dashed border. Below it, add a form to input metadata: Titre (required), Compositeur, Catégorie (dropdown), and Tags. Include a preview area that shows the PDF/Image once it's dropped. Implement a progress bar for the upload state. The design should be clean and minimize friction." 4. User Profile "Design an 'User Profile' dashboard. Include a header with the user's avatar, name, and membership status (Free/Premium). Display key statistics in small cards: 'Partitions uploadées', 'Lectures totales', 'Favoris'. Below, show a 'Recent Activity' list and a section to manage account settings (Email, Password, Notification preferences)." 5. Messe Page (Liturgical) "Generate a specialized 'Messe' route for ClefCloud. This page organizes partitions by liturgical parts (Entrée, Kyrie, Gloria, etc.). Use a 'Timeline' or 'Program' layout where a user can build a list of songs for a specific service. Include a 'Quick View' button for each song that opens the score in a mini-modal." 6. Premium Page "Design a 'ClefCloud Premium' pricing page. Show two tiers: 'Gratuit' and 'Premium'. Highlight the Premium tier with a subtle glowing border or a 'Populaire' badge. Use a checklist to compare features: Stockage illimité, Accès hors-ligne, Annotations avancées, Pas de publicités. Include a secure checkout form mockup using PayUnit styling."
